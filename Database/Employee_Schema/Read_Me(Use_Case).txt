										-------Employee Schema Scope-------

The purpose of this document is to describe the functionality and specifications of the design
of a web application for Managing Employees and their payroll.This system is designed for particular need of the company to carry out its 
operations in a smooth and effective manner.This web application is reduced as much as possible to avoid errors while entering data. It also 
provides error message while entering invalid data. It is user-friendly as no formal knowledge is required to use the system.This Employee Schema
Diagram consists of 6 parts which include Employee itself.Starting from Employee ,Each Employee is associated with each department,and each 
employee have particular salary .Also an Employee can be manager to a department ,so we have dep_manager to identify which employee is manager 
to which department. And also Employee has different titles to him/she .To conclude , everything is connected to the Employee Entity.
